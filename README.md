# 微信公众号 HTML 导出与排版规范

生成可直接粘贴到微信公众号编辑器的 HTML 文件，样式无损。

**作者**：里予

---

## 功能

- **一键复制** — 浏览器打开 HTML 后，点击按钮复制富文本到剪贴板
- **排版规范** — 标签白名单、CSS 安全属性，确保微信编辑器兼容
- **色号标准** — 正文 `#3f3f3f` / 标题 `#1a1a1a` / 次要文字 `#999999`
- **布局兼容** — 无 flex/grid/float/position，纯文档流
- **复查清单** — 11 项自动检查
