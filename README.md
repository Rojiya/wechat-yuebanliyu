# 月半里予公众号编辑技能

微信公众号「月半里予」的专属内容编辑技能，支持数据驱动的选题规划和文章撰写。

**作者**：里予

---

## 技能结构

```
wechat-yuebanliyu/
├── SKILL.md                          # 主技能（写作规则、风格内核、工作流程）
├── README.md                         # 本说明文件
├── wechat-html-export/
│   └── SKILL.md                      # 子技能（HTML导出、排版规范、色号标准）
├── scripts/
│   ├── read_archive.py               # 读取历史数据脚本
│   └── save_article.py               # 保存文章数据脚本（交互式）
└── 历史数据存档/
    ├── 数据汇总.json                  # 所有文章的核心数据
    ├── 文章历史/                      # 每篇文章的完整Markdown存档
    ├── 爆款标题库.json                # 高打开率标题存档
    ├── 读者反馈关键词.json             # 高频评论主题
    └── 迭代策略记录.md                # 每次数据复盘的策略调整
```

## 功能

### 主技能（wechat-yuebanliyu）

1. **数据复盘** — 每次启动新任务时读取历史数据，分析11项公众号指标
2. **智能选题** — 基于TLR质检框架生成3个候选选题
3. **文章大纲** — 输出结构、开篇方式、回环呼应设计
4. **全文撰写** — 按五种文章原型撰写，适配深夜树洞风格
5. **数据存档** — 发布后保存文章数据和内容到历史档案

### 子技能（wechat-html-export）

1. **HTML 生成** — 一键复制到公众号编辑器的完整 HTML 文件
2. **排版规范** — 标签白名单、CSS 安全属性列表
3. **色号标准** — 正文 `#3f3f3f` / 标题 `#1a1a1a` / 次要文字 `#999999`
4. **布局限制** — 微信编辑器兼容（无 flex/grid/float/position）
5. **复查清单** — 11 项输出后自动检查

## 使用方法

在对话中提及以下关键词即可触发：
- "准备写月半里予的新文章"
- "给我几个公众号选题"
- "分析一下上一篇文章的数据"
- "月半里予本周选题"
- "帮我写篇公众号文章"
- "导出HTML" / "生成HTML" / "公众号粘贴"

### 标准流程

1. 用户触发 → AI执行数据复盘（步骤0，索取上篇文章11项数据）
2. AI生成3个选题建议 → 用户选择
3. AI输出大纲 → 用户确认
4. AI撰写完整文章 → 用户审阅修改
5. 用户要求导出 → 调用子技能生成 HTML → 一键复制到公众号
6. 文章发布后用户提供数据 → AI存档

---

## 11项数据指标

阅读量、平均阅读时长、完读率、阅读后关注数、分享数、点赞数、收藏数、留言数、赞赏金额、推荐数、在看率

## 脚本使用

```bash
python scripts/save_article.py
```
